Portfolio: Material Design Lite theme for Hugo

Technology: web Link Screenshot:

As I wanted to learn more about Hugo while developing this blog, I created a new theme for Hugo. It is mostly plain HTML, CSS and JavaScript, but it also contains some Go Templating.

The theme is based on the blog template from Google’s Material Design Lite.

The source code is available on GitHub and I host a demo on GitHub Pages.
